======
avatar
======


Render the avatar img tag for a given backend user

Arguments
=========


backendUser (anySimpleType)
---------------------------


Default: 0

uid of the backend user

size (anySimpleType)
--------------------


Default: 32

width and height of the image

showIcon (anySimpleType)
------------------------


Default: false

show the record icon as well
